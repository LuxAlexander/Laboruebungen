# Prerequisites
* Python >= 3.6
* Poetry for dependency management
* pipx (to install poetry)
# Installation
* Download source
* pipx:https://github.com/pypa/pipx
* poetry: https://python-poetry.org/docs/
* Withing project directory tell poetry to fetch the dependencies  
`poetry install`

# Run demo program
* start program through poetry  
`poetry run python main.py`
